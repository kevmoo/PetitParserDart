json library

This package contains a complete implementation of JSON.

JsonParser creates a nested Dart objects from a given JSON string. For example the following code prints {a: 1, b: [2, 3.4], c: false}:

var json = new JsonParser();
var result = json.parse('{"a": 1, "b": [2, 3.4], "c": false}');
print(result.value);  // {a: 1, b: [2, 3.4], c: false}

The grammar definition JsonGrammar can be subclassed to construct other objects.
